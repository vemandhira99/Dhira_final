# Dhira - Enterprise Data Platform

Welcome to the DHIRA repository! This project contains the frontend codebase for the DHIRA platform, including the massive 100x architectural overhaul of the Akashic intelligence layer.

## Project Structure
- Built with **Next.js**, **React**, and **Tailwind CSS**.
- High-fidelity components located in `@/components/`.
- Custom geometric SVGs (Charminar, Enterprise Security) embedded for national-scale infrastructure motifs.
- Smooth scroll narratives via custom `ScrollReveal` hooks.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
